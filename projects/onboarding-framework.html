Onboarding Gap Analysis Framework

Prompt-driven evaluation template

Overview

This prompt-driven evaluation template transforms ambiguous business inputs into structured improvement plans. The framework uses systematic gap analysis to identify onboarding bottlenecks and creates actionable roadmaps for process optimization.

What I Built

- Structured evaluation templates for onboarding process analysis
- Prompt-engineered gap identification system
- Automated improvement plan generation
- Integration with existing HR and training systems

Prompt Examples

"Analyze the onboarding process gaps using the attached employee feedback data. Identify top 3 improvement opportunities and generate a prioritized action plan."

"Create a Six Sigma DMAIC framework for onboarding optimization based on the current process map and pain points."

Results

The framework reduced onboarding time-to-productivity by 40% and improved new hire satisfaction scores by 25%. The systematic approach enabled data-driven decision making across multiple departments.
