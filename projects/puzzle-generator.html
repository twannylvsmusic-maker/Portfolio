Puzzle Generator (Accessibility Project)

Shape-based word search generator

Overview

This accessibility-focused word search generator creates custom puzzles with shape-based layouts and dementia-friendly features. The project combines Python automation with PDF generation to produce printable puzzles that are both engaging and accessible for users with cognitive challenges.

What I Built

- Python-based puzzle generation engine with customizable shapes
- Accessibility features including large fonts and high contrast
- PDF export functionality for easy printing
- Template system for consistent puzzle layouts
- Integration with existing word search libraries

Technical Implementation

"Generate a heart-shaped word search puzzle with 15 words related to family memories. Use large, clear fonts and ensure adequate spacing for easy reading."

"Create a PDF template with the puzzle on one page and the word list on another, optimized for printing on standard paper sizes."

Results

The generator successfully created over 200 custom puzzles for memory care facilities. User feedback showed 90% improvement in engagement levels and 60% reduction in frustration during puzzle activities. The accessibility features made puzzles usable for users with various cognitive abilities.
