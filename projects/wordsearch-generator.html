Project Overview

This project showcases my full-stack development skills through a comprehensive word search puzzle generator. Built with Flask and modern web technologies, it demonstrates expertise in algorithm design, user interface development, and document generation.

The application features real-time puzzle preview, multiple shape options (including animals and objects), customizable themes and fonts, and professional PDF/Word export capabilities. It's designed with accessibility in mind and provides an intuitive user experience.

Puzzle Generation Algorithm

The core challenge was creating an efficient algorithm that could place words in various directions while respecting shape constraints and avoiding collisions.

def generate_puzzle(words, shape, allow_vertical=True, allow_horizontal=True, allow_diagonal=True):
    # Initialize grid based on shape
    grid = create_shape_grid(shape)
    
    # Sort words by length (longest first) for better placement
    sorted_words = sorted(words, key=len, reverse=True)
    
    placed_words = []
    for word in sorted_words:
        # Try multiple placement strategies
        if place_word_in_grid(word, grid, placed_words, allow_vertical, allow_horizontal, allow_diagonal):
            placed_words.append(word)
    
    return grid, placed_words
