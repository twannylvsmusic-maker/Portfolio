Fantasy Football Draft Forecasting

Using LLMs and statistical modeling to predict rookie success

Overview

This project integrates AI prompting, Next Gen Stats data, and combine metrics to forecast wide receiver performance. By analyzing key physical metrics, prospect grades, and prior year outcomes, I built a model that estimates rookie-year success probability and projected 2025 yardage.

What I Built

- Data preprocessing pipeline using Excel and ChatGPT Code Interpreter.
- Prompt-engineered evaluation system to identify outliers and correlations.
- Visual dashboards summarizing performance tiers across multiple seasons.
- Scenario forecasting to simulate draft outcomes based on combine data.

Prompt Examples

"Analyze the correlation between WR combine speed and rookie-year receiving yards. Include a linear regression summary and highlight top 5 overperformers."

"Using attached combine data, forecast 2025 WR projections exceeding 1000 yards and estimate success probability for Travis Hunter."

Results

The AI model achieved an R-value of 0.60 across multi-year data and accurately projected over 70% of WRs surpassing the 600-yard threshold. The interactive approach reduced manual correlation analysis time by 60%.
